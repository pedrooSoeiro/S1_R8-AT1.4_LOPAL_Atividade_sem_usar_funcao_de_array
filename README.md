# Atividade: Algoritmos e Estruturas de Vetores em JavaScript

Este repositório contém a resolução de três programas propostos para praticar a lógica de programação e a manipulação manual de estruturas de dados (vetores/arrays) em JavaScript, sem o uso de funções ou métodos prontos do ecossistema da linguagem (como `.push()`, `.filter()`, etc.).

## 🚀 Tecnologias Utilizadas

* **JavaScript (ES6+)** - Linguagem de programação utilizada no desenvolvimento dos algoritmos.
* **Git & GitHub** - Ferramentas utilizadas para o versionamento local e gerenciamento do repositório remoto.

---

## 📋 Regra de Negócio & Desafio Técnico

O principal objetivo desta atividade foi aprimorar a lógica de algoritmos através do controle manual de índices em vetores. Portanto:
* **Não foram utilizadas funções nativas de arrays** para adicionar ou filtrar elementos.
* Todo o controle de posições e iterações foi feito utilizando laços de repetição (`for` e `while`) e contadores manuais.

---

## 🛠️ Tarefas Desenvolvidas

### 1. Verificador de Maior Idade (`exercicio1.js`)
O programa lê a idade de 4 pessoas e as armazena em um vetor. Ao final da execução, exibe:
* Qual foi a maior idade digitada.
* A posição exata (índice) onde a maior idade foi inserida.

### 2. Filtro de Funcionárias (`exercicio2.js`)
O programa armazena os nomes, sexos e salários de 5 funcionários em três vetores distintos. Ao final, gera uma listagem exibindo apenas os dados das **funcionárias mulheres que recebem um salário superior a R$ 5.000**.

### 3. Análise Dinâmica de Alturas (`exercicio3.js`)
Este algoritmo permite que o usuário insira o nome e a altura de quantas pessoas desejar, utilizando uma estrutura de repetição condicional (`while`) que só encerra quando o usuário decide parar. 
Ao final, realiza uma busca para exibir:
* A maior altura cadastrada e o nome da respectiva pessoa.
* A menor altura cadastrada e o nome da respectiva pessoa.
